# 农业智能安全管理平台 (Python 3 & PyQt6 核心架构)

本目录包含完整的本地桌面端 **农业管理系统** PyQt6 源代码。项目采用全模块化架构，摒弃了传统的单文件面条代码，将十大核心业务模块分别封装在独立的文件中，在 `main.py` 主窗体中进行动态加载与信号绑定。

---

## 🛠️ 本地运行与环境部署

### 1. 初始化 Python3 虚拟环境 (venv)

在终端 (Terminal) 中切换到当前目录下，运行以下指令建立干净独立的运行环境：

```bash
# Windows 系统
python -m venv venv
venv\Scripts\activate

# macOS / Linux 系统
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装 PyQt6 依赖包

在激活的虚拟环境中，通过 `pip` 安全安装依赖：

```bash
pip install -r requirements.txt
```

### 3. 运行启动程序

在终端中执行以下指令，即可拉起 PyQt6 桌面客户端：

```bash
python main.py
```

## 🧠 核心业务设计矩阵 (每模块支持双引擎)

每个模块均已在代码中实现以下四大保障层中 **至少两种**（业务规则引擎、状态流转逻辑、调度/资源算法、数据一致性）：

1. **地块与作物管理**：
   - 规则引擎：连作科属隔离校验算法。
   - 状态流转：规划 -> 已播 -> 生长期 -> 已收获 自动推进。
2. **气象灌溉引擎**：
   - 规则引擎：气温与太阳辐射驱动的蒸发需水量估算。
   - 一致性：灌溉成功自动反馈更新土壤含水率。
3. **农资库存调度**：
   - 调度/算法：采购重订货点 (ROP) 公式。
   - 一致性：出库负值异常事务性回滚保护。
4. **无人机调度**：
   - 调度/算法：贪心求解多点植保最短路径（TSP）。
   - 规则引擎：遥测低电量限飞保护机制。
5. **智能测土施肥**：
   - 规则引擎：土壤 NPK 差额纯量化算公式。
   - 一致性：作业完工养分补足写回地块。
6. **生理积温虫控**：
   - 规则引擎：生理发育有效积温（GDD）叠加模型。
   - 状态流转：常态 -> 敏感潜伏 -> 中危 -> 爆发警报。
7. **采收计划与溯源**：
   - 规则引擎：有机物糖酸比成熟分级判别。
   - 一致性：采收生成唯一的数字物理溯源链哈希块。
8. **温室环控系统**：
   - 规则引擎：抗振荡双滞后死区控制回路。
   - 状态流转：加热/通风/气肥执行状态机跃迁。
9. **农事任务审批**：
   - 规则引擎：RBAC（系统管理员、经理、作业员）权限锁。
   - 状态流转：草稿 -> 审核 -> 批准 -> 执行 -> 完工。
10. **生产财务核算**：
    - 调度/算法：原材料与制造费用一键产出比例分摊。
    - 一致性：单据已分摊标志原子化更新，防止双重入账。
